# Scavenger Manifest Station ✨

> **全息能量显化与潜意识操控台**
>
> 这不是一个普通的 App，这是一个用于重塑思维频率、切断旧故事、强行对齐终点实相的潜意识接入终端。

## 核心功能模块
- **全息狂A注入舱**：支持 TXT 批量导入肯定语，通过高频物理重击（Haptic Feedback）将信念打入潜意识。
- **MD 潜意识防线**：内置暴力版、进阶版、情绪崩溃期版三大 MD 实操指引，实现思想的实时截断与覆盖。
- **圣多纳释放舱 2.0**：全内在沉浸式引导，剔除负面描述，直接剥离核心心魔。
- **宇宙灵感订单**：向宇宙后台发送实时订单，跟踪实相调配进度。

## 技术栈
- **Framework**: Flutter (Multi-platform)
- **UI Design**: Holographic Midnight Blue (全息午夜蓝)
- **Physics**: 3D Parallax Tilt (3D 视差倾斜)

## 运行与部署
1. 确保已安装 Flutter 环境。
2. `git clone https://github.com/你的用户名/仓库名.git`
3. `flutter pub get`
4. `flutter run`
